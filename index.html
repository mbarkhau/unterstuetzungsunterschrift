Datenschutzhinweis: Um die PDF Datei zu generieren werden Ihrer oben angegebene persönlichen Daten verarbeitet. Diese Verarbeitung findet ausschließlich auf Ihrem Endgerät statt. Eine Übermittlung oder Verarbeitung Ihrer persönlichen Daten an/durch andere Systeme findet nicht statt.

Gerne können Sie die Richtigkeit der Angaben zum Bürgeramt anfragen: Bürgerbüro anrufen. Fragen Sie dazu nach der "Zuständigkeit für die Bescheinigung des Wahlrechts für eine Unterstützungsunterschrift". Die zuständige Behörde führt dazu ein Wählerverzeichnis.

Sollte Ihrem Hauptwohnsitz keine Behörde, oder eine nicht zuständige Behörde zugeordnet werden, bitten wir Sie diesen Fehler auf Github zu melden.

Voraussetzungen für eine gültige Unterstützungsunterschrift

  • Sie müssen das 16. Lebensjahr vollendet haben.
  • Sie müssen deutscher Staatsbürger sein.
  • Sie dürfen nicht vom Wahlrecht ausgeschlossen sein.
  • Sie dürfen nur eine Unterstützungsunterschrift leisten. Sie dürfen nicht mehrere Parteien/Wahlvorschläge zur gleichen Wahl unterstützen.

Anleitung

  • Anforderungen
    1. Einen Drucker.
    2. Drei DIN A4 Seiten.
    3. Frankierter Briefumschlag (alternativ: selbst beim Amt abliefern).
  • Vorbereitung
    1. Füllen Sie unten Ihren Namen, Geburtsdatum und die Adresse von Ihrem Hauptwohnsitz aus.
    2. Drucken Sie die pdf Datei aus (idealerweise beidseitig).
    3. Falls Sie nicht beidseitig Drucken können, müssen Sie das Formblatt und den Datenschutzhinweis zusammenhefen.
    4. Prüfen Sie Ihre Angaben auf Richtigkeit. Wenn Ihre Daten nicht mit den bei Ihrer Gemeindebehörde hinterlegten Daten übereinstimmen, ist Ihre Unterstützungsunterschrift ungültig.
  • Unterschreiben
    1. Bekunden Sie Ihre Unterstützung, durch Ihre persönliche und handschriftliche Unterschrift auf dem Formblatt.
    2. Unterschreiben Sie auch den Datenschutzhinweis auf der Rückseite.
  • Versand an Ihre Gemeindebehörde
    1. Erstellen Sie den Behördenumschlag, adressiert an Ihre Gemeindebehörde.
    2. Legen Sie alle Ausdrucke in den Behördenumschlag.
    3. Versenden Sie oder überbringen Sie den Behördenumschlag an Ihre Gemeindebehörde.
  • Frankierung

    In der Regel übernimmt die Gemeindebehörde den Versand zur Sammelstelle. Sie können bei der Behörde gerne vorher anfragen, ob das der Fall ist. Sollte das nicht der Fall sein, legen Sie in den Behördenumschlag zusätzlich einen frankierten Antwortumschlag, adressiert an die Sammelstelle der Partei.
