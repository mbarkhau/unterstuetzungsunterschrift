UU-Tool

Anleitung und Automatisierung zur Erstellung von Unterstützungsunterschriften

Damit die Partei der Vernunft an der Europawahl 2024 antreten kann, müssen 4.000 bescheinigte Unterschriften bis zum 29.02.2024 gesammelt werden.

Mehr Infos zur Wahl finden Sie auf parteidervernunft.de/europawahl-2024/, sowie auf bundeswahlleiter.de/europawahlen/2024/.

Voraussetzungen für eine gültige Unterstützungsunterschrift

  • Sie müssen zum Wahltermin das 16. Lebensjahr vollendet haben.
  • Sie müssen die deutsche Staatsbürgerschaft besitzen.
  • Sie dürfen nicht vom Wahlrecht ausgeschlossen sein.
  • Sie dürfen nur eine Unterstützungsunterschrift leisten. Sie dürfen nicht mehrere Parteien/Wahlvorschläge für die gleiche Wahl unterstützen.

Anleitung

  • Anforderungen
    1. Einen Drucker.
    2. Drei DIN A4 Seiten.
    3. Zwei Briefumschläge.
    4. Mindestens eine € 0,85 Briefmarke (zwei, wenn Sie einen Gang zu Ihrer Gemeindebehörde persönlich übernehmen).
  • Vorbereitung
    1. Füllen Sie unten Ihren Namen, Geburtsdatum und die Adresse von Ihrem Hauptwohnsitz aus.
    2. Drucken Sie die pdf Datei aus (idealerweise beidseitig).
    3. Falls Sie nicht beidseitig Drucken können, müssen Sie das Formblatt und den Datenschutzhinweis zusammenhefen.
    4. Prüfen Sie Ihre Angaben auf Richtigkeit. Wenn Ihre Daten nicht mit den bei Ihrer Gemeindebehörde hinterlegten Daten übereinstimmen, ist Ihre Unterstützungsunterschrift ungültig.
  • Unterschreiben
    1. Bekunden Sie Ihre Unterstützung, durch Ihre persönliche und handschriftliche Unterschrift auf dem Formblatt.
    2. Unterschreiben Sie auch den Datenschutzhinweis.
  • Versand an Ihre Gemeindebehörde
    1. Erstellen Sie den Behördenumschlag, adressiert an Ihre Gemeindebehörde.
    2. Erstellen Sie den Antwortumschlag, adressiert an die Sammelstelle der Partei.
    3. Frankieren Sie den Behördenumschlag (es sei denn, Sie gehen persönlich zur Behörde).
    4. Frankieren Sie den Antwortumschlag (die Behörde übernimmt die Portokosten in der Regel nicht).
    5. Legen Sie alle Ausdrucke und den leeren Antwortumschlag in den Behördenumschlag.
    6. Versenden Sie oder überbringen Sie den Behördenumschlag an Ihre Gemeindebehörde.

Datenschutzhinweis:

  • Die Verarbeitung Ihrer persönlichen Daten findet ausschließlich auf Ihrem Endgerät statt.
  • Eine Übermittlung oder Verarbeitung Ihrer persönlichen Daten an/durch andere Systeme findet nicht statt.

Zuständige Behörde

-

Gerne können Sie die Richtigkeit der obigen Angaben anfragen: Bürgerbüro anrufen. Fragen Sie dazu nach der "Zuständigkeit für die Bescheinigung des Wahlrechts für eine Unterstützungsunterschrift". Die zuständige Behörde führt dazu ein Wählerverzeichnis.

Sollte Ihrem Hauptwohnsitz keine Behörde, oder eine nicht zuständige Behörde zugeordnet werden, bitten wir Sie diesen Fehler bei uns zu melden. Fehler per Email melden.
